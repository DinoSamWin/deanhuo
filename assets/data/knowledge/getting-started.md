# 欢迎来到我的知库
这里收集了我平时学习、思考的各类碎片。我在飞书写完文档后，直接将其**复制为 Markdown** 粘贴于此。

这套排版不仅支持基础的图文，还能自动识别视频和 Gamma！

## 1. 基础图文排版
你可以随意使用黑体、斜体、列表等基础功能：

- **深入理解产品**: 产品经理不仅是画原型的画图工，而是业务的引擎
- **技术的边界**: 了解技术是为了更好地定义产品的边界
- **摄影的力量**: 光影记录着我们生活过的痕迹
